1. <style> 태그
→ <head> 태그 내부에 사용
→ 태그명, 클래스명, 아이디, 요소 선택자에 스타일 적용
→ <img /> 태그의  src 속성과 같이 class="{클래스명}", id="{아이디}" 방식으로 사용

// <style> 태그 내부
p {         } -> 태그명으로 접근
.{클래스명} {            } -> 클래스명으로 접근
#{아이디명} {            } -> 아이디로 접근

※ id -> html && 태그에서 유일해야함 (1개만 있어야 됨)
※ class -> 여러 번 중복해서 사용할 수 있음

// Font 관련 스타일
- color : 폰트 색상 변경 (색상 문자열, 16진수의 RGB값 등)
- font-size : 폰트 사이즈 변등 (단위 : px, ...)
- font-weight : 폰트 굵기 지정
- font-family : 폰트 변경
※ font-family: '맑은 고딕', sans-serif, Arial
→ '맑은 고딕'으로 변경, 없으면 sans-serif로 변경, 없으면 Arial로 경경
- text-decoration : 글자 꾸미기

// 부피, 여백 관련 스타일
- width : 태그의 너비 지정 (단위 : px, %, ...)
- height : 태그의 높이 지정 (단위 : px, %, ...)
- background-color : 배경색 지정
- background-image : 배경에 이미지 지정 (단, 이미지와 CSS가 포함된 파일이 동일함 경로에 존재해야 함)
- border : 태그의 경계선 지상
※ border: {굵기} {종류} {색상}
※ border 종류 : solid, dotted, dashed, ...
- padding : 태그 내부의 여백 지정 (단위 : px, %, ...)
- margin : 태그 외부의 여백 지정 (단위 : px, %, ...)
※ border, padding, margin 속성
-> {속성}-top, {속성}-bottom, {속성}-left, {속성}-right로 부분적으로 적용 가능

// Display 관련 스타일
- display : none, block, inline, inline-block 방식 등 존재
※ display : none; -> 화면에 나타나지 않음
※ display : block; -> 화면에 block 형식으로 나타남 (별도로 지정하지 않을 경우, 화면 전체 너비에 맞춰 조절됨)
※ display : inline; -> 태그 안의 요소들의 크기만큼 inline 형식으로 나타남
※ display : inline-block; -> inline과 block의 특징을 합침

// Position
- position : absolute
-> 화면의 top-left (= 0, 0)을 기준으로 위치 선정
- position : relative
-> 화면의 정해진 특정 위치를 기준으로 위치 선정
